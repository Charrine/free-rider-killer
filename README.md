# Tieba Manager

贴吧管理工具

http://www.tiebamanager.xyz

## 使用方法

### 删贴机

`python2 TiebaAutoTool.py delete`

### 封禁机

`python2 TiebaAutoTool.py block`

请使用-h参数来查看帮助
